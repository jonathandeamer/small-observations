# scripts/_archive/

Holding pen for one-off sweep scripts that completed their purpose and aren't worth running again, but are kept around as references for similar future work (audit shape, frontmatter parsing patterns, etc.).

**Nothing in here is wired into `make check` or the Makefile.** If a script earns ongoing value, promote it back to `scripts/` and wire it in.

If a script in here would be more confusing than useful to keep, just delete it — git history is the long-term archive.
